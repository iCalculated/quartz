# [Hugo](https://gohugo.io)

## [Frontmatter](https://gohugo.io/content-management/front-matter/)

Title is required.

## Hugo Obsidian 

Used to generate graph and interactive links:

```shell
hugo-obsidian -input=content -output=data
```

## Run

Can server locally with

```shell
hugo server
```
